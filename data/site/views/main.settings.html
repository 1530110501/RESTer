Inspections for requests
RESTer can analyze your request as you type and point out possible mistakes. Currently this includes:
  • Using placeholders without having variables enabled.
  • Using a variable with an empty value.
Strip default headers
RESTer uses the browser to send requests, which will add a couple of default headers to requests like Accept, Accept-Encoding or User-Agent. When you enable this option, RESTer will strip these headers unless they are explicitly provided in a request.
